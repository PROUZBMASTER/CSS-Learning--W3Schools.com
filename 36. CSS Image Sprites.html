CSS Image Sprites

Image Sprites

An image sprite is a collection of images put into a single image.
A web page with many images can take a long time to load and generates multiple server requests.
Using image sprites will reduce the number of server requests and save bandwidth.

Example explained:-

  • <img id="img1" src="Image19.gif"> - Only defines a small transparent image because the src attribute cannot be empty. The displayed image will be the background image we specify in CSS.But we did not include the src attribute which is not recommended. Because the image will have no placeholder in that location. But specifying src attribute with non-existing image will place a non-transparent image placeholder in that location which will interfere with background image.
  • width:46px;height:44px; - Defines the portion of the image we want to use
  • background:url('Image19.gif') 0 0; - Defines the background image and its position (left 0px, top 0px)

Image Sprites - Create a Navigation List

We want to use the sprite image ("Image19.gif") to create a navigation list.
We will use a HTML list, because it can be a link and also supports a background image.
Example:-


Example explained:-
  • #navlist {position:relative;} - position is set to relative to allow absolute positioning inside it
  • #navlist li {margin:0;padding:0;list-style:none;position:absolute;top:0;} - margin and padding are set to 0, list-style is removed, and all list items are absolute positioned
  • #navlist li, #navlist a {height:44px;display:block;} - the height of all the images is 44px


Now start to position and style for each specific part:-
  • #li1 {left:0px;width:46px;} - Positioned all the way to the left, and the width of the image is 46px
  • #li1 {background:url('Image19.gif') 0 0;} - Defines the background image and its position (left 0px, top 0px)
  • #li2 {left:63px;width:43px;} - Positioned 63px to the right (#li1 width 46px + some extra space between items), and the width is 43px
  • #li2 {background:url('Image19.gif') -47px 0;} - Defines the background image 47px to the right (#li1 width 46px + 1px line divider)
  • #li3 {left:129px;width:43px;} - Positioned 129px to the right (start of #li2 is 63px + #li2 width 43px + extra space), and the width is 43px
  • #li3 {background:url('Image19.gif') -91px 0;} - Defines the background image 91px to the right (#li1 width 46px + 1px line divider + #li3 width 43px + 1px line divider)
