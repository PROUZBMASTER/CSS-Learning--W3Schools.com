The var() Function Example

Sometimes we want the variables to change only in a specific section of the page.
Assume we want a different color of blue for buttom elements. Then, we can re-declare the --blue variable inside the button selector. When we use var(--blue) inside this selector, it will use the local --blue variable value declared here.
We see that the local --blue variable will override the global --blue variable for the button elements.
